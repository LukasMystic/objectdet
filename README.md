# ObjectDet - Adaptive Knowledge Transfer Protocol

An AI-powered object detection system featuring a novel Adaptive Knowledge Transfer Protocol (AKTP) distillation method.

## Features

- **Dual-Teacher Distillation:** ResNet18 + EfficientNet-B0 teaching a lightweight Student.  
- **Hybrid Architecture:** Deep Learning feature extraction with SVM classification.  
- **Glassmorphic UI:** React + TailwindCSS + Framer Motion.  
- **FastAPI Backend:** Asynchronous inference and MongoDB integration.
